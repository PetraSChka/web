== Description == 
Данный плагин предназначен для упрощения интеграции wordpress и QIWI по средством WooCommerce.

Модуль добавляет способ оплаты Ваших заказов.

Вы сможете:
- Выставлять счета
- Принимать оплату
- Производить возвраты
- Получать актуальные статусы счетов

Ваши клиенты смогут производить оплату с помощью банковских карт или киви кошелька на их усмотрение.

Для использования плагина , необходимо зарегистрироваться на [b2b.qiwi.com](https://b2b.qiwi.com) (для юридических лиц и ИП) или на [p2p.qiwi.com](https://p2p.qiwi.com)  ( для физических лиц и самозанятых).

Так же вы можете ознакомиться с примером работы формы - [Демо](https://developer.qiwi.com/demo/ru/)

Возможности протокола, используемого для создания плагина описаны в  [Документации](https://developer.qiwi.com/ru/bill-payments/)



== Installation  ==
Минимальные требования
* Woocommerce 3
* PHP 7 witch cURL extension

Для начала необходимо установить актуальную версию плагина WooCommerce ( можно скачать [тут](https://ru.wordpress.org/plugins/woocommerce/))
Далее  войдите в вашу консоль WordPress, перейдите в раздел меню плагинов и нажмите ‘Добавить новый’.
В поле поиска введите «QIWI касса» и нажмите Enter. 
Найдя наш плагин , для установки, просто нажмите  «Установить».
